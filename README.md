# PlayStation Fan Website 🎮

An unofficial, non-commercial fan-made website inspired by PlayStation's design.
Built as a front-end practice project.

## 🛠️ Tech Stack
- HTML5
- CSS3 + Tailwind CSS
- JavaScript (Vanilla)
- Swiper.js (carousel)
- Font Awesome (icons)
- Animate.css

## ✨ Features
- Responsive navigation with dropdown
- Hero section with animated text
- Game showcase with video hover preview
- Interactive Swiper carousel
- PlayStation Plus pricing cards
- PS5 stats counter section
